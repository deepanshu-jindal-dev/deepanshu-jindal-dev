## 🚀 What I'm working on

- 🔨 **ChatDocs** — Full-stack AI app where users chat with their PDFs. Built on a microservices architecture with a Python FastAPI RAG microservice, Node.js/Express backend, MongoDB Atlas, and React frontend. Includes JWT auth, per-user data isolation in Pinecone, and rate limiting.
- 🧠 **CS Knowledge Engine** — Production-grade Hybrid RAG pipeline with dynamic alpha-weighted hybrid search (Gemini dense + BM25 sparse), Cohere reranking, multi-turn query reformulation, and a full RAGAS evaluation suite.
